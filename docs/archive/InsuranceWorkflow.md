# Background

This is an Insurance Management System supporting multiple tenant types.

Current tenant types are:

```php
broker
underwriter
```

Currently the system uses a common Policy Creation workflow which automatically generates policy numbers.

This is incorrect for brokers.

Insurance Brokers do NOT issue policies.

Insurance Underwriters issue policies.

Brokers place risks with underwriters and later record policy details issued by the underwriter.
